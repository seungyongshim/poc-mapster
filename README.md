### 1. Ports
* 포트는 어플리케이션 코어의 경계에 존재한다.
* 포트는 외부세계의 언어와 도메인 언어를 모두 알고 있다.
* 동일한 도메인에서 재활용 가능하다.

### 2. Adapter
* 외부세계의 관심사만을 처리한다.
* 포트는 어뎁터의 언어를 알고 있다.
* 다른 비즈니스에서 재활용 가능하다.
